Supervised Learning Tutor Bot (DTI5125/Group 26

This chatbot guides users through key supervised learning concepts such as labels, features, classification, regression, data splitting, evaluation metrics, and common issues like overfitting and underfitting.

What this bot can do

The bot is designed as an interactive tutor. It can both guide the user through a supervised learning workflow and answer side questions during the conversation.

  • Explain supervised learning concepts in simple language
  • Guide the user from label identification to model evaluation
  • Suggest models for classification and regression tasks
  • Explain metrics like accuracy, precision, recall, F1-score, MAE, MSE, and RMSE
  • Diagnose issues such as overfitting and underfitting
Labels Features Classification Regression Train/Test Split Metrics Overfitting Underfitting

Try questions like: “What is a label?”, “Explain regression”, “How do I evaluate my model?”, or “My training accuracy is high but test accuracy is low.”

How to use it

Open the chat widget in the bottom-right corner and start a conversation. You can either ask direct conceptual questions or let the bot guide you step by step.

  • Start with: I want to learn supervised learning
  • Ask side questions anytime during the tutoring flow
  • Continue naturally after each explanation
